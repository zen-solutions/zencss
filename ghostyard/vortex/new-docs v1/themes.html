Themes

ZenCSS offers a range of theme customization options to provide a unique look and feel for your UI. Themes are defined under the $themes map and can be customized further with the provided classes.

Defining Themes:

Define themes using the $themes map and specify colors for each theme.


$themes: (
  primary: $light-color,
  dark: $light-color,
  secondary: $secondary-color
);

Theme Classes:

Use theme classes to apply styles based on the defined themes.


.theme-primary {
  background: $primary-color;
  color: $light-color;
}

.theme-dark {
  background: #505050;
  color: #f4f4f4;
}

.theme-secondary {
  background: $secondary-color;
  color: #f4f4f4;
}
/* ... and so on for other themes ... */

Accent Classes:

Apply accent styles to elements within a theme.


.theme-dark.accent-left {
  border-left: 10px solid $primary-color;
}

.theme-secondary.accent-left {
  border-left: 10px solid $cta-color;
}
/* ... and so on for other accent classes ... */

Form Styles within Themes:

Style form elements based on the theme applied.


.theme-light form label {
    color: $light-color;
}

.theme-dark button {
    background: $light-color;
    color: $dark-color;
}
/* ... and so on for other form styling within themes ... */
