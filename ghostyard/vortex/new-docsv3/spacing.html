Spacing Utilities

zenCSS provides spacing utility classes to easily set padding and margin values on elements. The classes range from .p-0 to .p-7 for padding, and .m-0 to .m-7 for margin. Additionally, you can target specific sides using classes like .mt-0 to .mt-5 for margin-top and .pt-0 to .pt-5 for padding-top, and so on.

Padding Classes


  .p-0 { padding: 0; }
  .p-1 { padding: 0.25rem; }
  .p-2 { padding: 0.5rem; }
  .p-3 { padding: 0.75rem; }
  .p-4 { padding: 1rem; }
  .p-5 { padding: 1.25rem; }
  .p-6 { padding: 2em; }
  .p-7 { padding: 3em; }

Margin Classes


  .m-0 { margin: 0; }
  .m-1 { margin: 0.25rem; }
  .m-2 { margin: 0.5rem; }
  .m-3 { margin: 0.75rem; }
  .m-4 { margin: 1rem; }
  .m-5 { margin: 1.25rem; }
  .m-6 { margin: 2em; }
  .m-7 { margin: 3vw; }
