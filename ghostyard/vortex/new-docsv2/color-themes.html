Color Themes

The Color Themes section introduces a mechanism to easily switch between different color schemes in your project. These themes affect various UI elements including forms, buttons, and more. The themes are defined using a Sass map and are applied using classes.

Theme Definition

Themes are defined in the $themes map. Each theme has a name and associated color. Hereâ€™s an example of how themes are defined:


$themes: (
  primary: $light-color,
  dark: $light-color,  // Assuming you have a $dark-color variable
  secondary: $secondary-color  // Assuming you have a $secondary-color variable
);

Applying Themes

Themes are applied using classes. The class name is prefixed with .theme- followed by the theme name. For example, .theme-primary, .theme-dark, and .theme-secondary.

Form Label Styling

Form labels are styled based on the applied theme:


@each $theme, $color in $themes {
  .theme-#{$theme} {
    form label {
      color: $color;
    }
  }
}

Accent Border

Accent borders can be applied on the left side of elements using the .accent-left class along with the theme class. Here are examples of how these classes are defined:


.theme-dark.accent-left{
  border-left:10px solid $primary-color;
}

.theme-secondary.accent-left{
  border-left:10px solid $cta-color;
}

.theme-cta.accent-left{
  border-left:10px solid $secondary-color;
}

.theme-light.accent-left{
  border-left:10px solid $accent-color;
}

Background and Text Color

Background and text colors are styled based on the applied theme:


.theme-light {
  background: rgba(255, 255, 255, 0.4);
  color:$dark-color;
  border: $accent-color;
}

.theme-dark {
  background: #505050;
  color: #f4f4f4;
}

.theme-secondary {
  background: $secondary-color;
  color: #f4f4f4;
}

.theme-accent {
  background: $accent-color;
  color: $dark-color;
}

.theme-cta {
  background: $cta-color;
  color: #f4f4f4;
}

.theme-primary {
  background: $primary-color;
  color: $light-color;
}
