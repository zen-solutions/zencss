zenCSS Icons - Companion Class Generator

Use this tool to generate companion CSS classes to colorize zenCSS icons

zenCSS icons are black but you may want to colorize them for your website. Since they are SVG files, you can't apply normal CSS color to them, you need to specify a filter. This tool will generate a CSS class for you similar to, and compatible with your source color.

Step 1: Enter your desired color in hex format.

target
render

Step 2: Copy the CSS and use for your icons

 .your-icon { }

This tool is alpha. Rerun as needed. This may not work for all values. Use a slightly different value if you are getting inconsistent results.

For this code to generate a valid class the icon color needs to be black. If your icon set isn't black you can prepend "brightness(0) saturate(100%)" to your filter property which will first turn the icon set to black.
