Dive into a world where color, shape and form come together seamlessly, and everything falls into place to keep the vibes balanced.

Quick Overview—The zenCSS Smart-Align System

Think of containers like individual chunks of your webpage - like this big blue section.

They're all about keeping your layouts tidy and aligned, no matter if someone’s checking out your site on a mobile, tablet, or desktop.

Rows are the building blocks within a container, stackable like layers on a cake. Add as many or as few as you need—our Smart-Align System makes sure they display beautifully.

Rows are full width and hold columns like this one right here, which are basically supercharged HTML buckets.

Columns are where the magic happens.

Whether showcasing a section heading and paragraph or a themed layout, they’re up for it.

Columns auto-adjust and auto-format to display beautifully across all devices and screen sizes. Here, they act as themed cards, marrying form and function.

Putting zenCSS into action

Kick things off by downloading or cloning the main branch, then navigate to docs -> examples -> hello-world.html. Or check out our sandbox here. Once you're up and running, you’re just a container and row declaration away from wrapping your HTML content in ZenCSS goodness.

Here's a basic example using semantic HTML. 

<main class="z-container">
  <section class="z-row">
    <div class="z-col"> ... </div>
  </section>
</main>

If you're already using Bootstrap, zenCSS integrates seamlessly into your workflow, providing enhanced capabilities.

The beauty of ZenCSS is in its go-with-the-flow nature. Whether you're packing a row with 2 or 6 columns, or filling a column with waaay too many things, ZenCSS handles the layout with grace. Want to jazz up a content area? Just toss in more rows or columns.

If you're all about keeping things snappy, say hello to Zentax—our sleek shorthand HTML syntax that'll slash your coding effort in half: 

<z-container>
  <z-row>
    <z-col> ... </z-col>
  </z-row>
</z-container>
 Note: Our toolkit includes a user snippet to generate the above code in 5 keystrokes, and a program to convert Zentax into semantic HTML.

The zenCSS Smart-Text System

Automatic, dynamic, responsive resizing

zenCSS comes with a dynamic text scaling feature that takes the guesswork out of font sizing across varying viewport resolutions, ensuring readability and aesthetic consistency are always in check.

Right out of the gate, our standard paragraph text is sized as follows:

  • 15px on mobile
  • 16 px on tablet
  • 18 on desktop screens

The heading tags (h1 - h6) are cleverly set up to resize at each breakpoint, based on the default font size. For a closer look, inspect an element below and switch to the computed tab in your browser's developer tools while toggling the responsive view. Controlling how content breaks

At the row level, zenCSS is all about ensuring a graceful display and realignment of items across devices. Yet, if you fancy some extra control, our utility classes are here to let you fine-tune the layout. By default, zenCSS displays four columns on desktop, two on tablet, and one on mobile. With six children in a row, tossing the class two-up to the row will have each child cozying up to half the width. Similarly, throw in three-up and each child will claim a third of the space. Simple, elegant spacing for your elements.

At the column level, we use Flexbox logic in our Smart Align System to stack, position and align HTML elements within columns, making them act like more like a mini-collection of rows. To place items side by side, just add a div as a parent container. This way, it's children can break free from the default flex settings of the z-col container and sit next to each other. This simple step helps easily control the layout and alignment of your elements within a z-col element.

Debugging Stylistic Issues chrome extension available

Debugging CSS can be difficult with mobile/tablet variations, pre-processors, absent source maps, and minification in the mix.

Enter "Zen Test" in zenCSS—a handy tool that overlays outlines on content, making bug identification and resolution a lot more intuitive. It pairs well with your browser's inspect element tool, simplifying the task of isolating issues.

Mobile troubleshooting becomes a breeze when you can quickly pinpoint where bugs are cropping up.

Outlines draw the element boundaries with blue dotted lines for containers (purple for -fluid), rows (green), and columns (red), while paragraphs and headings sport varying shades of orange outlines. An alignment grid is brewing on our end.

Manual Installation

Get zenCSS up and running on your site by linking to our CSS file in the HTML head section, and including the zencss.js file just before the closing body tag. 

<link rel="stylesheet" href="path-to/zen.css">
<script src="path-to/zencss.js"></script>

Usage

Deploy zenCSS in two ways: globally across the site, or selectively on specific HTML elements.

Globally: Declare zen on your body to apply zenCSS site-wide. 

<body class="zen"> 
...
</body>

Mix in: Invoke zen on any HTML element to harness zenCSS's power locally, without disrupting existing elements. 

<section class="zen"> 
...
</section>
While working inside a parent element declared as zen, you're what we refer to as "in zen", granting access to all zenCSS tools and features.
