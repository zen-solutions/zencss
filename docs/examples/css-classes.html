zenCSS modifier classes

With modifier classes, tailoring the appearance of HTML elements is a breeze.

To showcase their utility, let's tweak the code example UI element on this site to adopt the brand theme.

<pre class="default-primary">...</pre>

In zenCSS, themes and background classes format both background and text colors of their children, but the flexibility doesn't end there. You are able to override both attributes at the container, row, column or element level as needed, or use our text classes to override the text color of any element.

From a design standpoint, consider using this formula for element styling: theme → background modifications → text modifications → other styling, animations, and so on. Review our ZEN Ideology for a better idea of how our system is structured.
