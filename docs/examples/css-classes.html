zenCSS modifier classes

With over 100 modifier classes, tailoring the appearance of HTML elements is a breeze.

To showcase their utility, let's tweak the code example UI element on this site to adopt the brand theme.

<pre class="default-primary">...</pre>

Themes influence both background and text colors, yet the flexibility doesn't end there. We want you to be able to override either aspect at the element level as needed. The styling vision is as follows: theme → background modifications → text modifications → other styling, animations, and so on—all without the need for !important. We're not there yet, but we're hard at work to make it happen.
