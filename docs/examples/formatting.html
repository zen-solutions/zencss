Formatting Utilities

Formatting utilities in zenCSS allow for a wide range of styling adjustments pertaining to dimensions, alignment, and more. Below are the classes available under formatting utilities:

Gap Classes

These classes are used to set the gap property for an element, ranging from .gap-0 (no gap) to .gap-6 (0.6em gap).

.gap-0 { gap: 0; }
.gap-1 { gap: 0.1em; }
.gap-2 { gap: 0.2em; }
.gap-3 { gap: 0.3em; }
.gap-4 { gap: 0.4em; }
.gap-5 { gap: 0.5em; }
.gap-6 { gap: 0.6em; }

Height Classes

These classes set a specific height on an element, ranging from .height-1 (10em) to .height-6 (40em).

.height-1 { height: 10em; }
.height-2 { height: 15em; }
.height-3 { height: 20em; }
.height-4 { height: 25em; }
.height-5 { height: 30em; }
.height-6 { height: 40em; }

Width Classes

These classes set a specific width on an element, ranging from .width-1 (4rem) to .width-5 (15rem).

.width-1 { width: 4rem; }
.width-2 { width: 6rem; }
.width-3 { width: 9rem; }
.width-4 { width: 12rem; }
.width-5 { width: 15rem; }

Directional Modifiers

These classes adjust the margin of an element in specific directions. The classes range from .up-10 to .right-50, incrementing by 10.

.up-10 { margin-top: -10px; }
.down-10 { margin-top: 10px; }
.left-10 { margin-left: -10px; }
.right-10 { margin-right: 10px; }
.up-50 { margin-top: -50px; }
.down-50 { margin-top: 50px; }
.left-50 { margin-left: -50px; }
.right-50 { margin-right: 50px; }

Width and Height Classes

These classes set specific width and height values on an element, ranging from .w-10 to .w-500 and .h-10 to .h-500.

.w-10 { width: 10px; }
.h-10 { height: 10px; }
.w-500 { width: 500px; }
.h-500 { height: 500px; }
