Color Classes

Unique Color Classes:

.bg-opaque: Background color with opacity (20% opaque white).

.bg-blur: Applies a blur effect to the backdrop.

.text-dark-green: A text color class for dark green.

.highlight: Highlights text with a background color of yellow.

.text-black: A text color class for black.

.icon-invert: Inverts the color of icons.

.icon-gold and .icon-silver: Apply color filters to create gold and silver color effects on icons.
