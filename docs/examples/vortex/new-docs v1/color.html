Color

The Color section in zenCSS comprises various classes that allow you to easily style the background, text, and border colors of HTML elements. These classes are based on the predefined color maps outlined in the User Settings section.

Background Color Classes

These classes are used to set the background color of an element.


    .bg-primary { background-color: $primary-color; }
    .bg-secondary { background-color: $secondary-color; }
    .bg-accent { background-color: $accent-color; }
    .bg-cta { background-color: $cta-color; }
    .bg-red { background-color: $red; }
    .bg-orange { background-color: $orange; }
    .bg-yellow { background-color: $yellow; }
    .bg-green { background-color: $green; }
    .bg-blue { background-color: $blue; }
    .bg-purple { background-color: $purple; }
