User Settings

In ZenCSS, user settings are organized under various categories to allow for easy customization. These settings include toggling testing mode, icon usage, setting font families, defining color palettes, specifying border radius, and more. Here's a breakdown of the user settings:

Initial Settings:

  • Testing Mode: Toggle testing mode on or off with $testing-mode: true or $testing-mode: false.
  • Site Icons: Enable or disable site icons with $site-icons: true or $site-icons: false.
  • Font Family: Set the font family with $font-family: font-family-list.

Color Palettes:

  • Brand Color Palette: 
    
  $primary-color : #1f77d5;
  $secondary-color: #16426c;
  $accent-color: hsl(43, 86%, 53%);
  $cta-color: #de4842;
  • Generic Color Palette: 
    
  $red: #E63946;
  $orange: #ff9653;
  $yellow: #e4c257;
  $green: #0fbd74;
  $blue: #3c8ceb;
  $purple: #9B59B6;
  • Alternative Color Palette: 
    
  $red: #FF0000;
  $orange: #FF7F00;
  $yellow: #FFFF00;
  $green: #00FF00;
  $blue: #0000FF;
  $purple: #8B00FF;

Dark and Light Colors:

  • $text-white: #eee;
  • $text-black: #111;
  • $light-color: #f4f4f4;
  • $dark-color: #424242;
  • $grey: #eee;
  • $dark-grey: #666;
  • $light-grey: #ccc;
  • $bg-default: $dark-color;
  • $text-default: $light-color;

Rounded Corners:

  • Set the border radius with $border-radius: 4px;

Site Images:

  • Set the hero image with Modifiers
    : url("../website/images/hero.jpg");

Experimental:

  • $light-ui-hover-color:#eee;

Default Spacing:

  • $site-section-padding-top: 50px;
  • $site-section-padding-bottom: 50px;
  • $container-margin: 2em;
  • $col-padding: 1em;
  • $col-margin: 1em;
