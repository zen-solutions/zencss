Typography

The Typography section in zenCSS houses classes for text formatting, alignment, size, and special characters. It's designed to provide a straightforward way to style text elements in your project.

Text Formatting

Style the font weight and style of your text with the following classes:


.italic { font-style: italic; }
.bold { font-weight: 700; }
.semi-bold { font-weight: 500; }

Text Alignment

Align text and block-level elements with these classes:


.center { text-align: center; width: 100%; align-items: unset; }
.v-center { margin: auto; width: 100%; }
.center-flex { display: flex; justify-content: center; align-items: center; }
.right { text-align: right; }
.left { text-align: left; }

Text Size

Adjust the size of text with the following classes:


.small { line-height: 1.4em; font-size: .8em; }
.smaller { font-size: .6em; }
.big { font-size: 1.2em; }
.bigger { font-size: 1.4em; }

Special Characters

Style special character elements using the below class:


.tout-circle {
    width: 20px;
    height: 20px;
    border-radius: 50%;
    background: $green;
    color: white;
    font-size: 14px;
    text-align: center;
    line-height: 20px;
    position:absolute;
    top:19px;
    right:19px;
    padding-left:0 !important;
    padding-right:0 !important;
}

Icons

Define the size of icon fonts with the below class:


.icon-lg { font-size: 23px; }
