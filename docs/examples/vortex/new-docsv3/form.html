Form Elements

The Form Elements section provides classes and mixins to style various form elements. Utilizing these classes will help maintain consistency in form styling across your project.

Input Styles Mixin

Apply common styles to input fields, textareas, and select elements with the input-styles mixin:


@mixin input-styles {
    width: 100%;
    padding: 5px;
    border: 1px solid $light-color;
    border-radius: $border-radius;
    color: dark-color;
    outline: none;
}

Button Styles Mixin

Style buttons with the button-styles mixin:


@mixin button-styles {
    padding: 10px 20px;
    border: none;
    border-radius: $border-radius;
    background-color: $cta-color;
    color: $light-color;
    font-size: 16px;
    cursor: pointer;
    outline: none;

    &:hover {
        background: darken($cta-color, 10%);
    }
}

Form Structure

Structure your form with labels, input fields, and buttons. The classes provided allow for easy styling of form elements:


form {
    margin: auto;
    padding: 1em;
    padding-top:10px;

    fieldset {
        margin-bottom: 20px;
        padding: 20px;
        border: none;
    }

    p {
        margin-bottom:1em;
    }

    label {
        display: block;
        margin-top: 2px;
        color: $primary-color;
    }

    input[type="text"],
    input[type="password"],
    input[type="number"],
    input[type="email"],
    input[type="url"],
    input[type="tel"],
    input[type="search"],
    input[type="date"],
    input[type="datetime"],
    input[type="datetime-local"],
    input[type="month"],
    input[type="week"],
    input[type="time"],
    input[type="color"],
    input[type="range"],
    textarea,
    select {
        @include input-styles;
    }

    input[type="submit"],
    input[type="reset"],
    input[type="button"],
    button {
        @include button-styles;
    }
}

Form Footer

Style the footer of your forms with this class:


.form-footer {
    font-size: .7rem;
    display: block;
    margin-top: 1em;
    opacity: .5;
}

Shrinking Form Labels

Shrink and move labels on input focus or if there is content in the input field:


form div {
    position: relative;
    margin-bottom: 20px;
}

form input {
    width: 100%;
    padding: 5px;
    border: 1px solid $border-color;
    color: $dark-color;
}

.form-container form label {
    position: absolute;
    left: 8px;
    top: 7px;
    font-size: .8em;
    color: $dark-color !important;
    transition: 0.3s all;
}

form input:focus + label,
form input:not(:placeholder-shown) + label {
    top: -1px;
    left:6px;
    font-size: 9px;
    color: #b9b9b9 !important;
}
