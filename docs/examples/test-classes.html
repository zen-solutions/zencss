Test Classes

These classes are designed for on-the-fly troubleshooting, especially useful during the development phase to highlight elements on the page.

  • .test-1: Adds a dotted outline around the element with a green color.
  • .test-2: Adds a solid outline around the element with a red color.
  • .test-3: Adds a solid outline around the element with a blue color.
  • .test-4: Adds a solid outline around the element with an orange color.
  • .test-5: Adds a solid outline around the element with a purple color.

Utilizing these test classes makes it easier to visually identify elements and their dimensions on the page, aiding in debugging and layout adjustments.
