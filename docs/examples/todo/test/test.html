Sweet material tabs

Just playing around with tabs and material design.

To use the tabs we need a wrapper data-tabs, the tabs data-tabs-tab and the indicator data-tabs-indicator. If You want, you can add data-tabs-active to the default active tab.
Markup should be like this: 

    <nav data-tabs>
      <div>
        <a data-tabs-tab data-tabs-active>Tab One</a>
        <a data-tabs-tab>Tab Two</a>
        <a data-tabs-tab>Tab Three</a>  
      </div>
      <span data-tabs-indicator>
    </nav>

Then we add a bit of CSS to the data-tabs and data-tabs-indicator: 

    [data-tabs]{
      position: relative;
    }
    [data-tabs-indicator]{
      position: absolute;
      left: 0px;
      bottom: 0px;
      height: 2px;
      width: 0px;
      background-color: rgb(255,87,34);
      -webkit-transition: all .5s;
      transition: all .5s;
    }

Add this jQuery function to your app: 

    
    /**
     * jTabs
     *
     * @url https://codepen.io/wolf-w/pen/MwGzbj/
     * @author Wolf Wortmann <http://elementcode.de> / <wolf@wolfgang-m.de>
     * @copyright (c) Copyright 2015 Wolf Wortmann <http://elementcode.de> / < wolf@wolfgang-m.de>
     * @license Feel free to use, modify and redistribute this code. But keep this license AND copyright notice. Backlinks to my page are nice c:
     */
    ;(function($) {
      $.fn.jTabs = function() {
        $.each(this, function(){
          var $this = $(this),
              $caaler = $this,
              $tabs = $('[data-tabs-tab]', $this),
              $indicator = $('[data-tabs-indicator]', $this);
          jQuery.fn.jTabs.update = function(){
            var $this = $('[data-tabs-active]');
            if(!$this.length){return false;}
            var data = {
                  'w' : $this.outerWidth(),
                  'x' : $this.position().left
                };
            $indicator.css({
              'width' : data.w+'px',
              'left' : data.x+'px'
            });
          }
          $caaler.jTabs.update();
          
          jQuery.fn.jTabs.change = function($this){
            $tabs.removeAttr('data-tabs-active');
            $this.attr('data-tabs-active', true);
            $caaler.trigger('tabChange', [{
              '$element' : $this,
              '$tabs'   : $tabs,
              '$indicator': $indicator,
              '$caaler'   : $caaler
            }]);
            $caaler.jTabs.update();
          }
          
          $tabs.click(function(event){
            event.preventDefault();
            event.stopPropagation();
            $caaler.jTabs.change($(this));
          });
        });
      }
    }(jQuery));

Now it's time to start all the things and relax the rest of the day..: 

    $(document).ready(function(){
      $('[data-tabs]').jTabs();
    });

If you want more than a switching indicator, you could listen to the tabChange event and play around c: 

    $(document).ready(function(){
      $('[data-tabs]').jTabs();
      $('[data-tabs]').on('tabChange', function(event, data){
        $trigger = data.element;
        //load data via ajax or make some other cool things
        console.log('load page '+$trigger.attr('href'));
      });
    });

You can also manually update the indicator by jTabs.update() or trigger one tab manually by jtabs.change($newTab).
$newTab should be a (jQuery) data-tabs-tab element (in this example case one of the a tags). 

    //change the data-tabs-active attribute to another elment
    //now call the jTabs.update()
    $('[data-tabs]').jTabs.update();
    
    //we want to change the current tab to the last tab
    $('[data-tabs]').jTabs.change($('a[data-tabs-tab]:last-child'));
