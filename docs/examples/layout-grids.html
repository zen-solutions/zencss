Layout Grids

Create any layout you can imagine with our tools. Here are some example layout grids using our framework, the concept of nesting containers in rows (instead of a column), and our modifiers.

Large Hero + 4 across

 hero col 1 2 3 4
Sample Code 
<z-container>
    <z-row>
        <z-col class="bg-grey-3 h-300">hero col</z-col>
    </z-row>
    <z-row>
        <z-col class="bg-grey-3">1</z-col>
        <z-col class="bg-grey-3">2</z-col>
        <z-col class="bg-grey-3">3</z-col>
        <z-col class="bg-grey-3">4</z-col>
    </z-row>
</z-container>

        <p>large content area</p>
        </z-col>
        <z-container>

            <z-col class="bg-grey-3">1</z-col>
            <z-col class="bg-grey-3">2</z-col>
        </z-row>
        <z-row>
            <z-col class="bg-grey-3">long col</z-col>
        </z-row>
        <z-row>
            <z-col class="bg-grey-3">long col</z-col>
        </z-row>
        <z-row class="pb-0">
            <z-col class="bg-grey-3">long col</z-col>
        </z-row>
        </z-container>
    </z-row>
</z-container>

Multi-section Left Panel (Container replaces left z-col)

 hero col 1 2 3

large content area
Sample Code 
<z-container>
    <z-row>
        <z-container>
            <z-row class="pt-0">
                <z-col class="bg-grey-3 h-225">hero col</z-col>
            </z-row>
            <z-row class="pb-0">
                <z-col class="bg-grey-3">1</z-col>
                <z-col class="bg-grey-3">2</z-col>
                <z-col class="bg-grey-3">3</z-col>
            </z-row>
        </z-container>
        <z-col class="bg-grey-3">
            <p>large content area</p>
        </z-col>
    </z-row>
</z-container>

note: If you are setting height, set it for 1 col only and let the rest flow. Above i set it for hero col
