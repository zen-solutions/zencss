Text formatting, alignment and size

zenCSS provides a robust set of text formatting utilities, enabling you to style and align text with ease. Whether you are aiming for a bold statement or subtle emphasis, these classes have got you covered.

Text Formatting Classes

.italic: Applies italic style to text.

.thinnest - This text is the thinnest.

.thinner - This text is thinner.

.thin - This text is thin.

.regular - This text is regular.

.medium - This text is medium.

.semibold - This text is semi-bold.

.bold - This text is bold.

.bolder - This text is bolder.

.boldest - This text is the boldest.

Text Alignment Classes

.center: Centers text horizontally within its container.

.center-vertical: Centrally aligns the element vertically.

.right: Aligns text to the right within its container.

.left: Aligns text to the left within its container.

Text Size Classes

.smaller: Decreases font size to 0.6em.

.small: Sets the font size to 0.8em.

.big: Increases font size to 1.2em.

.bigger: Increases font size to 1.4em.

Usage Examples

Here are some examples showcasing how you might use these utilities in your HTML: 


                    <!-- Example 1: Apply italic style to text -->
                    <p class="italic">This text is italic.</p>
            
                    <!-- Example 2: Center align text -->
                    <p class="center">This text is center aligned.</p>
            
                    <!-- Example 3: Set font size to bigger -->
                    <p class="bigger">This text has a font size of 1.4em.</p>
