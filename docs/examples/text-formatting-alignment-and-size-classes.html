Text formatting, alignment and size

Text Formatting Classes

.italic: Applies italic style to text.

.bold: Makes text bold with a font weight of 700.

.semi-bold: Applies a semi-bold font weight of 500 to text.

Text Alignment Classes

.center: Centers text horizontally within its container.

.center-vertical: Centrally aligns the element vertically.

.center-flex: (Not defined, possibly for centering content using Flexbox.)

.right: Aligns text to the right within its container.

.left: Aligns text to the left within its container.

Text Size Classes

.small: Sets the font size to 0.8em.

.smaller: Decreases font size to 0.6em.

.big: Increases font size to 1.2em.

.bigger: Increases font size to 1.4em.

.serif: Sets the font family to 'Playfair Display', serif.
