Text formatting, alignment and size

zenCSS provides a robust set of text formatting utilities, enabling you to style and align text with ease. Whether you are aiming for a bold statement or subtle emphasis, these classes have got you covered.

Text Formatting Classes

.text-thin - This text is thin.

.text-regular - This text is regular.

.text-medium - This text is medium.

.text-semibold - This text is semi-bold.

.text-bold - This text is bold.

.text-bolder - This text is bolder.

.text-boldest - This text is the boldest.

.text-italic: Applies italic style to text.

Text Alignment Classes

.text-center: Centers text horizontally within its container.

.text-center-vertical: Centrally aligns the element vertically.

.text-right: Aligns text to the right within its container.

.text-left: Aligns text to the left within its container.

Text Size Classes

.text-smaller: Decreases font size to 0.6em.

.text-small: Sets the font size to 0.8em.

.text-big: Increases font size to 1.2em.

.text-bigger: Increases font size to 1.4em.

Span Classes

.text-superscript: Displays text as superscript. For example, the dollar sign here $49

.label: Displays text as a label. ie NEW

.eyebrow: Displays text as a callout above content areas. ie NEW

.text-dual: Switches text to light color for dark mode only. Useful for colored text on the default background. text-primary text-dual

Usage Examples

Here are some examples showcasing how you might use these utilities in your HTML: 


                    <!-- Example 1: Apply italic style to text -->
                    <p class="italic">This text is italic.</p>
            
                    <!-- Example 2: Center align text -->
                    <p class="center">This text is center aligned.</p>
            
                    <!-- Example 3: Set font size to bigger -->
                    <p class="bigger">This text has a font size of 1.4em.</p>
