Flex

Flex Utilities

Leverage the power of the Flexbox layout model with zenCSS's flex utilities. Effortlessly create complex layouts, align elements, and control the dimensions of your flex items with these intuitive classes.

1. Flex Container Class

  • .d-flex: This class sets the display property to flex, making the element a flex container.

2. Flex Grow Classes

These classes are used to set the flex-grow property of flex items, which determines how much a flex item will grow relative to the rest of the flex items in the flex container.

  • .feature: flex-grow is set to 2
  • .feature-sm: flex-grow is set to 3
  • .feature-md: flex-grow is set to 5
  • .feature-lg: flex-grow is set to 6

Developer Notice

Be advised, the .feature-lg class is defined twice. This issue will be resolved in the upcoming release.

Usage Examples

Here are some examples showcasing how you might use these utilities in your HTML: 


                    <!-- Example 1: Create a flex container -->
                    <div class="d-flex">
                        <div class="feature">Item 1</div>
                        <div class="feature-sm">Item 2</div>
                        <div class="feature-md">Item 3</div>
                        <div class="feature-lg">Item 4</div>
                    </div>
