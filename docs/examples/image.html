Image Styles

Enhance your images with these easy-to-apply classes to fit the needs of your layout and design.

Round Image

Shapes the image into a perfect circle. The element must have equal width and height dimensions to create a circle rather than an ellipse.

<img src="path-to-image.jpg" class="img-round" alt="">

Round Image with Outline

Gives the image a full circular shape with a solid white outline and a subtle drop shadow for depth. The image must have equal width and height for the outline to appear even.

<img src="path-to-image.jpg" class="img-round-outline" alt="">

Rounded Image

Applies a small, uniform border-radius to create rounded corners on an image.

<img src="path-to-image.jpg" class="img-rounded" alt="">

Left-aligned Image

Floats the image to the left of the parent element and applies margins for spacing around the image.


<img src="path-to-image.jpg" class="img-left" alt="">

Right-aligned Image

Floats the image to the right of the parent element and applies margins for spacing around the image.

<img src="path-to-image.jpg" class="img-right" alt="">

Responsive Image

Makes an image scale nicely to the parent element. The image will scale down if it has to, but never scale up to be larger than its original size.

<img src="path-to-image.jpg" class="img-responsive" alt="">
