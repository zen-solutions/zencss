Outlined Buttons

In need of a button, but without the hefty background colors?

Ensure sufficient background contrast when using outlined buttons.
Code Example 
<button class="btn-primary-outline">Primary</button>
<button class="btn-secondary-outline">Secondary</button>
<button class="btn-cta-outline">CTA</button>
<button class="btn-accent-outline">Accent</button>
<button class="btn-dark-outline">Dark</button>
<button class="btn-light-outline">Light</button>
