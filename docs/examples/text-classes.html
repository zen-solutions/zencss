Sans-serif

Noto Sans is the default font applied throughout the application, and doesn't require a specific class to be applied. "Noto" means "I write, I mark, I note" in Latin. The name is also short for "no tofu", as the project aims to eliminate 'tofu': blank rectangles shown when no font is available for your text.
