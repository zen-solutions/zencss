Using zenCSS & Bootstrap together

zenCSS and Bootstrap can be used interchangeably, allowing developers to leverage the strengths of both frameworks in a project. To demonstrate, we'll load zenCSS followed by Bootstrap to give Bootstrap preference in the CSS cascade, and construct a common layout using both frameworks.

Transitioning Between zenCSS and Bootstrap

By closing the parent element with the zen class, you can switch to using Bootstrap's default styling and spacing. For instance, after transitioning, you'll notice our default styling and spacing is gone, demonstrating the seamless shift between the two frameworks.

Bootstrap example layout

In this example, we'll utilize Bootstrap to create a layout with a container encompassing a single row, which in turn houses three columns. These columns are designed to stack three abreast on desktop, two on tablet, and one on mobile. Adjusting the browser width demonstrates the responsive behavior.

<div class="container">

  </div>
</div>

zenCSS Equivalent Layout

Having transitioned back to zenCSS, let's resolve the same layout challenge using zenCSS's container, row, and column framework. Similar to the Bootstrap example, the goal is to have a container housing a single row with three columns. They should stack in the same manner based on the device screen size. Moreover, zenCSS's solution accommodates adding or removing items without necessitating any additional code modification.

Lorem Ipsum Dolor

Here's how our solution was written:

<z-container>
  <z-row>
    <z-col class="bg-blue">Lorem</z-col>
    <z-col class="bg-green">Ipsum</z-col>
    <z-col class="bg-red">Dolor</z-col>
  </z-row>
</z-container>

zenCSS envisions itself as a complementary tool, usable alongside other frameworks without the need for overriding CSS with !important. Although still in beta, zenCSS aims to provide a hassle-free, code-efficient solution for responsive web design.
