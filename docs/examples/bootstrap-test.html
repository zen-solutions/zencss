Building a typical web layout in Bootstrap

Let's use bootstrap to build a test using their container, row, col framework. The goal, have a container housing one row, which houses 3 columns. They must stack 3 up on desktop, 2 up on tablet and 1 up on mobile. You can change the width of your browser window to see it working.

Now we will move back "in zen", and take a look at what was written.

Here's how their solution was written:

<div class="container">
  <div class="row">
    <div class="col-lg-4 col-md-6 col-12 bg-primary">Lorem</div>
    <div class="col-lg-4 col-md-6 col-12 bg-success">Ipsum</div>
    <div class="col-lg-4 col-md-12 bg-danger">Dolor</div>
  </div>
</div>

Let's solve for the same issue in zenCSS

Here we'll use zenCSS build a test using our container, row, col framework. The goal, have a container housing one row, which houses 3 columns. They must stack 3 up on desktop, 2 up on tablet and 1 up on mobile. As a bonus, our solution will account for adding or removing items without needing to modify or write any additional code.

 Lorem Ipsum Dolor

Here's how our solution was written:

<z-container>
  <z-row>
    <z-col class="bg-blue">Lorem</z-col>
    <z-col class="bg-green">Ipsum</z-col>
    <z-col class="bg-red">Dolor</z-col>
  </z-row>
</z-container>

This is our vision for zenCSS. To be a tool you can use alongside other frameworks, when and where it makes sense for you without ever needing to resort to writing !important again. We're still in beta though so deploy to production with caution.
