Using zenCSS & Bootstrap Together

zenCSS and Bootstrap can be used interchangeably, allowing developers to leverage the strengths of both frameworks in a project. To demonstrate, we'll load zenCSS followed by Bootstrap to give Bootstrap preference in the CSS cascade, and construct a common layout using both frameworks.

Transitioning Between zenCSS and Bootstrap

By closing the parent element with the zen class, you can switch to using Bootstrap's default styling and spacing. For instance, after transitioning, you'll notice our default styling and spacing is gone, demonstrating the seamless shift between the two frameworks.

Bootstrap example layout

In this example, we'll utilize Bootstrap to create a layout with a container encompassing a single row, which in turn houses three columns. These columns are designed to stack three across on desktop, two on tablet, and one on mobile. Adjusting the browser width demonstrates the responsive behavior.

Here's how their solution was written:


<div class="container">
  <div class="row">
    <div class="col-lg-4 col-md-6 col-12 bg-primary">Lorem</div>
    <div class="col-lg-4 col-md-6 col-12 bg-success">Ipsum</div>
    <div class="col-lg-4 col-md-12 bg-danger">Dolor</div>
  </div>
</div>

it's worth noting, to add or remove columns you need to adjust the col classes to allocate space for the columns that will be in the final composition.. Not the case with zenCSS.

zenCSS Equivalent Layout

Let's resolve the same layout challenge using zenCSS's container, row, and column framework. Similar to the Bootstrap example, the goal is to have a container housing a single row with three columns. They should stack in the same manner based on the device screen size. zenCSS's solution automatically accommodates adding or removing items without the need to update col-x classes, as well as WCAG-compliance.

Lorem Ipsum Dolor

Here's how our solution was written:

<z-container>
  <z-row>
    <z-col class="bg-blue">Lorem</z-col>
    <z-col class="bg-green">Ipsum</z-col>
    <z-col class="bg-red">Dolor</z-col>
  </z-row>
</z-container>

A hassle-free, code-efficient solution for responsive web design - zenCSS can be used in a standalone capacity or alongside other frameworks.
