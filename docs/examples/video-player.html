Responsive Video Player

A simple, responsive wrapper for self-hosted or YouTube videos.

Code Example 
<div class="video-container">
    <video controls  poster="path_to_your_thumbnail.jpg">
        <source src="path_to_your_video.mp4" type="video/mp4">
        <source src="path_to_your_video.webm" type="video/webm"> <!--optional fallback video-->
        Your browser does not support the video tag.
    </video>
</div>
Code Example 
<div class="youtube:video-container">
    <iframe src="https://www.youtube.com/embed/2JvMga7bPB0" frameborder="0" allowfullscreen></iframe>
</div>
