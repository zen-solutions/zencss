Button Styles and Sizes

These classes are tailored for styling and sizing buttons.

Button Classes:

.btn

.btn-ghost

.btn-ghost-light

.btn-ghost-dark

.btn-invert

Button Sizes:

.btn-xl: Extra-large button with width: 200px;

.btn-lg: Large button with width: 150px;.

.btn-md: Medium button with width: 120px;.

.btn-sm: Small button with width: 75px;.

Unique Button Styles:

.btn-square: A square button with a 3D effect and arrow indicators.

.inactive: An inactive state style with cursor: not-allowed;, opacity: 0.65;, and pointer-events: none;.
