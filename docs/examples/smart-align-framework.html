The zenCSS Smart-Align Framework

Dive into a world where shape and form align, and everything magically falls into place.

Overview

The zenCSS Smart-Align System uses a custom Flexbox grid system to provide you with an advanced, responsive layout solution for your web apps. Like most modern CSS frameworks' grid systems, it is composed of three main components - containers, rows and columns:

  • The container is the outermost layer of the grid system and serves as a nestable, responsive wrapper. Containers are used as the basis for your layout.

  • Inside the container, you can add one or more rows , which will in turn house the columns.

  • Columns are placed inside rows and they can be customized to display differently on different screen sizes, make use of fluid layouts, use features and change ordering.

Advanced Features

By default, the Smart-Align System stacks four columns on desktop, two on tablet and one on mobile. You can modify this behavior, but the system is dynamic and will adjust to any amount of columns automatically. Odd columns size up accordingly. Resize the window to see it in action.

You may use "x-up" prefix to specify modifier classes that change the default alignment (where x is a number from two through eight). here we add the class eight-up to the row. We also target mobile with the .mobile-two-up class so devices smaller than tablet will display columns side by side.

One more thing.. If you want to go full screen, you'd use container-fluid. Note: We use a "z-" prefix for container, row and column class names.

<div class="z-container">
  <div class="z-row">
    <div class="z-col">

    </div>
