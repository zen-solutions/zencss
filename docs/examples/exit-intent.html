Exit Intent

A popup that activates when a user intends to leave the page.

Wait 8 seconds (configurable) then move your mouse towards the url bar to trigger the exit intent. Exit intent will be hidden for 7 days (configurable) upon exiting the offer. This is done via a flag set in local storage.
