The zenCSS Smart-Color System

Universally accessable color is important. That's why The zenCSS Smart-Color System adheres to WCAG 2.1 AA best practices. Each background color defaults to a font color with sufficient contrast for use with normal text, large text and graphics in both light and dark mode. Verify here.
